Simple key codes in sucro programs:

 {R}             ENTER key (also terminates a line in sucro listing)
 {home}          HOME key
 {pgdn}          Page Down key (next page)
 {pgup}          Page Up key (previous page)
 {erase}         ERASE (ctrl-END) key
 {ins}           INSERT key
 {del}           DELETE key  {del3} same as {del}{del}{del}
 {ins line}      Insert line key (alt-F9)
 {del line}      Delete line key (alt-F10)
 {act}           ESC key (activate Survo command)
 {r}             right arrow  {r3} same as {r}{r}{r}
 {l}             left arrow   {l3} same as {l}{l}{l}
 {u}             up arrow     {u3} same as {u}{u}{u}
 {d}             down arrow   {d3} same as {d}{d}{d}
 {(} {)}         characters { and }
 {sp}            <space key> 
 {tab}           TAB key
 {line end}      END key
