NAMES=<1 or 0> 
(in PLOT operation for bar and pie charts) specifies names of the
observations to be plotted for each bar/pie.
Default is NAMES=1, i.e. the names will be plotted.
By NAMES=0, the names are omitted.
For example, on the screen (GPLOT), NAMES=[RED],1 implies that the
names are written in red.

NAMES=<#_of_lines>,<line_space> 
(in PLOT operation for vertical bar charts of type VBAR etc.)
specifies the names of the observations to be plotted on several
lines using a given line space.
