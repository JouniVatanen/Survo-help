Getting information from the Web by F1 key combinations

F1 G      calls Google by the current keyword(s).
F1 S      calls Google by the current keyword(s) and "Survo".
F1 W      calls Wikipedia by the current keyword(s).
F1 D      calls the Free Online Dictionary by the current keyword(s).

Before pressing the above F1 key combinations the cursor must
  touch the keyword in the edit field, or
  touch the space between a combination of two keywords.
A sequence of several keywords can also be painted by the rightmost
mouse button.

The default settings for key codes above can be changed by
SURVO.APU lines of the following form

F1-code     SURVO.APU
F1 g        google=http://www.google.com/search?q= 
F1 W        wiki=http://en.wikipedia.org/wiki/ 
W1 D        dict=http://www.thefreedictionary.com/ 


User-defined F1 codes for Web links

Ten various F1 Web settings can be set by lines
web0, web1, web2,..., web9 corresponding to codes
F1 0, F1 1, F1 2,..., F1 9

For example, activating the command

SYSTEM web1=http://www.youtube.com/results?search_query= 

gives access to YouTube pages in the current Survo session.
Then by touching the word Sibelius in the edit field and by F1 1
we get hundreds of links to videos where music of Sibelius is played.
The same YouTube setting will be created permanently by the command

SYS    web1=http://www.youtube.com/results?search_query= 



User-defined F1 codes for Web links (continued)

These settings can be defined even more accurately by giving, for
example,

web7=http://www.google.com/search?q=||survo||+||%22 
where || works as a delimiter for the following items:
http://www.google.fi/search?q=    address of the form http://
survo                             additional keyword
+                                 separator of keywords
%22                               "quotation mark"

"quotation mark" is a string of characters surrounding the sequence
of keywords. %22 is the inherent notation for it in Google.
The defaults are: no additional keyword, separator is '+', no
"quotation mark".
