LIST operations are used for processing of long lists and reports.
Such lists consist of several chapters specified by DEF definitions
(See DEF?) in different edit files.

  1 = LIST definition 
  2 = Structured LISTs 
  3 = LIST SHOW 
  4 = LIST REPLACE      Replacing a string by another in the list 
  5 = LIST SORT         List sorting 
  6 = LIST SAVE         Saving list fields by cases in a text file 
  7 = LIST MAKE         Creating lists from text files 
  8 = LIST COUNT        Frequencies of given phrases (strings)
