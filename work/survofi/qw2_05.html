In the case of unequal sample sizes there are no uniformly best
way to form hypotheses. Therefore, the user may select one of
the three choices in the ANOVA operation:

- If HFORM=0 is specified then tests are based on sequential sums
of squares.
- If HFORM=1 is specified then hypotheses are chained according
to the "natural testing order" of the effects (default).
- If HFORM=2 is specified then tests are formed using sigma-
restricted models (regression method).
