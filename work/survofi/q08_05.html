A control line (with a '-' in the control column) in the print list
and having the form
  - text <name_of_text_file>,L1,L2
specifies lines L1 to L2 from an ASCII file to be included in the set
of lines to be printed. If L1 and L2 are omitted the entire file will
be printed. If L2 is omitted only, lines from L1 up to the end of file
are printed.
