General information on sucros

Any Survo session or a part of it can be saved in a special file
(with extension .TUT) as a sucro by working in the tutorial mode.
This file can be later activated and it imitates actions normally taken
by the user.
A sucro file can be loaded to the edit field (TUTLOAD operation),
edited and saved back to the sucro file (TUTSAVE operation).
Then various extra comments, conditional statements, prompts for the
user and time delays may be inserted, too.
The sucros in Survo range from small macros to user-defined operations
and large expert applications.

Originally, the tutorial mode was planned for making tutorials and
therefore many technical terms still have 'TUT' as a distinctive part.
