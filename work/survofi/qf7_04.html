Aggregation of observations in a Survo data file

There are two operations for this task, FILE AGGRE and FILE AGGR
with different options. Since they both require using of other
FILE operations, the easiest way for typical aggregations is provided
by the sucro /AGGRE which combines FILE AGGR with other necessary
actions automatically.

  1 = /AGGRE 
  2 = FILE AGGRE 
  3 = FILE AGGR
