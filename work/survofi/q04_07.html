Key codes in SHOW:
ENTER=Exit  N or PgDn=Next page  P or PgUp=Previous page
(right arrow)=Scroll to the right stepwise
(left  arrow)=Scroll to the left  stepwise
END=Scroll to the right (one screen width)
HOME=Scroll to the left (one screen width) / To the start of the file
E=To the end of the file (Interrupt by `.')
L=Load selected lines to the edit field (Interrupt by `.')
C=Append selected lines to another text file.
S=Search a string or by number of line (#3000) (Interrupt by `.')

In text files:
D=Edit the first line in the window
d=Edit the bottom line in the window
   In editing, parts of text may be overwritten.
   Neither lines nor characters can be inserted or deleted.
W=Convert Windows characters into Survo characters (ISO1->IBM8)
   See also LOADW? SAVEW?
