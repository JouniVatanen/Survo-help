XLABEL=<a_string_without_spaces> 
(in PLOT operation for diagrams and curves) gives a title for the X
axis. Default is the name of the variable etc. depending on the type of
the graph.
YLABEL=<a_string_without_spaces> 
(in PLOT operation for diagrams and curves) gives a title for the Y
axis. Default is the name of the variable etc. depending on the type of
the graph.
  P = more information on plotting
