The current data path is shown at the end of the header line of
the Survo main window. Thus all files on this path can be referred to
simply by their names without a path designation.

The data path can be changed by the CD command and it may be changed
automatically during the Survo session when the user makes selections
from various menus, etc.

When the data path is on the system path of Survo, the part of the
system path is designated simply as <Survo>. It is strongly recommended
that the user creates for his/her tasks own data paths which are
not on the <Survo> path. The <Survo> path is used by the Survo programs,
other system files, and demo files.
Thus notation <Survo> can be used only in certain basic and technical
commands of Survo.

  1 = CD (and DISK) command and other ways of changing the data path 
  2 = Commands accepting <Survo> notation for the system path 
  3 = F4:DISK key
