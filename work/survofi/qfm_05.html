Various derived fields (calculated according to rules of editorial
computing) are presented on FILE MEDIT pages in the form

Population density: ####.# inhabitants/km^2 / density=Popul/Area

giving in data FINLAND for Helsinki:

Population density: 2731.1 inhabitants/km^2 / density=Popul/Area

Values of variables and expressions can be presented also
as data-dependent comments, for example:

Population density in %1 is %2. / %1=Commune %2=round(Popul/Area,1)

giving in data FINLAND for Helsinki:

Population density in Helsinki is 2731.1.
