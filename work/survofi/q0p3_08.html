The EPSFILE specification is not available in SURVO MM.
To convert a PostScript file into an Encapsulated PostScript file use
EPS <file.PS>,<file.EPS> after PLOT!

A Survo graph made by PLOT can be saved in a file in the EPS format
by including the specifications
  DEVICE=PS,<PSfile>  EPSFILE=<EPSfile> 
in the PLOT scheme.
Then a Survo PostScript file <PSfile> is first generated and
then the EPS command of Survo is automatically called to
convert <PSfile> into Encapsulated PostScript format as <EPSfile>.
Any <PSfile> can be taken in the PRINT list by - picture line
and any <EPSfile> by - epsfile line.
(The latter alternative is essential for EPS files imported from
other sources.)
