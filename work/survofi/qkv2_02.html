Processing of dates with the DATE operation                  / K.Vehkalahti

The DATE operation includes several functions for processing of dates
given in the edit field or, more generally, in the Survo data sets.

A plain DATE command without parameters gives the date and time in the form
DATE / Wednesday January 24 1996 22:48:22 Week=04 Day=024
on the current (DATE) line.
