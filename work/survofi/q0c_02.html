CODES LOAD <binary_file>  / BYTES=<N> 
loads <N> first characters of <binary_file> to the edit field
and lists them on consecutive lines below the current line.
Default value for <N> is BYTES=256.
The list of the characters has the following form:
CODES LOAD SORTCODE.BIN_
     0     0
     1     1
     2     2
     3     3
    ..    ..    .  .
    97    65    a  A
    98    66    b  B
    99    67    c  C
    ..    ..    .  .
where the second column gives the decimal code of the corresponding
character in the file. (For example, in SORTCODE.BIN, which is used
in alphabetic sorting, 97 is mapped to 65 (i.e. 'a' is like 'A').
Columns 3 and 4 are for explanations only.

  C = More information on CODES operations
