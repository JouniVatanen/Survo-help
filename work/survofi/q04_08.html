SAVEP <text file> 
saves the edit lines below the current line in <text file>. The former
contents of <text file> will be destroyed. The various display
attributes (underlining) are not saved. To save parts of the edit field
to the end of <text file> use COPY.

If the text file will be used in another Windows program,
use the SAVEW command instead of SAVEP.
SAVEW has the same options and it makes the code conversions automatically.

If the text file will be used in Linux or Unix programs,
use the SAVEU command instead of SAVEP.

SAVEP L1,L2,<text file> 
saves edit lines L1-l2 in <text file>.

To load text saved by SAVEP use LOADP or SHOW.
