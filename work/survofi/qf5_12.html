TXTEDTOUT <edt_file>,<ascii_file> 
copies an edit file (edit field in an .EDT file)
to an ascii (text) file. Also the control column and
shadow lines are copied.
The ascii file can then be sent through a network
to another Survo installation where it can be transformed
back to an edit file by the command
TXTEDTIN <ascii_file>,<edt_file> .

If only text (neither the control column nor shadow lines)
is to be sent, the standard SAVEP command should be used.
