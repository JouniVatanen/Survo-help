Classical multidimensional scaling (Richardson 1938, Torgerson 1952)

/CSCAL D,m,CSCAL,CSEIGEN,CSCENT,CSDIST
makes classical MDS from a given distance matrix D.

Thus D is a n*n matrix of distances or dissimilarities of
n objects.
m gives the number of dimesions (scales) to be computed.

The results are saved in matrix files as follows:
                                              default matrix name
Scale values (n*m)                     CSCAL          CSCAL.M
Eigenvalues (1*n)                      CSEIGEN        CSEIGEN.M
Eigenvalues (percentages) (1*n)        CSCENT         CSCENT.M
Reproduced distances (n*n)             CSDIST         CSDIST.M

   M = More information on MDS
