trim_width=<integer> (in SURVO.APU)
tells the default line width for TRIM commands.
Default is trim_width=72 (traditional width of the Survo main window).

trim_tolerance=<integer> (in SURVO.APU)
gives the shortest permitted length of the start of a hyphenated word
at the end of the line when using the TRIM3 command.
Default is 2. Other suitable values are 3,4,...
