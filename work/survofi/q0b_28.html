New SURVO MM system parameters in SURVO.APU:

  1 = videomode    (Logical dimensions of a graph) 
  2 = gplot_layout (Locations and sizes of Survo graphics windows) 
  3 = soft_keys    (Soft keys below the SURVO MM main window) 
  4 = edit_font, help_font, os_font (Sizes and fonts of SURVO MM windows) 
  5 = printer      (Default PostScript printer) 
  6 = cursor       (Size of the cursor) 
  7 = help_window  (using a separate window for help texts) 
  8 = trim_width,trim_tolerance (for TRIM commands) 
  9 = close_graphs (closing Survo graphics windows on exit) 
  A = sucro_speed  (default speed in tutorials) 
  B = language     (working language) 
  C = session_tmp  (control of subdirectories for temporary files) 

  S = General information on Survo system parameters
