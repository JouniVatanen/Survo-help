The probabilities p0,p1,p2,... of a discrete distribution of a variable
with possible values 0,1,2,... can also be given by a FIT specification
of an alternative form
FIT=MATRIX(<name_of_matrix_file>)   ( for example, FIT=MATRIX(P5) )
In this case the probabilities p0,p1,p2,... must be saved as the first
column of the matrix file.
If these probabilities have been computed by using some estimates
of parameters (computed from the current data), the number of
such parameters should be given by a PARAM specification.
