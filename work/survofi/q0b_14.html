Window size:
The size of the visible part of the edit field (the window size) is
determined by parameters `er3' and `ec3'. `er3' is # of lines and `ec3' #
of columns. Example: er3=23 ec3=72 (traditional values)
During a Survo session the window size can be changed by
RESIZE <er3>,<ec3>.

The physical size (fonts) and locaction of the Survo Main window
is changed most conveniently by soft buttons SYSTEM -> WINDOW.
