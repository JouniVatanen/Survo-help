  A = - chapter 
  B = - text 
  C = - picture

  F = - define 
  G = - shadow 
  H = - control 
  I = - codes 
  J = page changes 
  K = effects of display attributes (shadows) 
  M = device drivers 
  N = directing the output to a file or to another printer 
  O = - dos 
  P = footnotes 
  Q = - index 
  R = - ascii_text 
  S = miscellaneus control words 
  W = control words in brackets ([line_spacing(x)],[page_length[x], etc.) 
  X = General information on PRINT
