Function keys when ALT is pressed at the same time:
+----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+
| F1 |  | F2 |  | F3 |  | F4 |  | F5 |  | F6 |  | F7 |  | F8 |  | F9 |  | F10|
|    |  |    |  |    |  |    |  |    |  |    |  |    |  |    |  |    |  |    |
+----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+
SOFT    WORDS   COPY    BLOCK   SEARCH  FILE    CODE      -     LINE    LINE
BUTTONS                                    ACT                     INS     DEL

  1 = ALT-F1:SOFT BUTTONS     2 = ALT-F2:WORDS 
  3 = ALT-F3:COPY LINE        4 = ALT-F4:BLOCK 
  5 = ALT-F5:SEARCH           6 = ALT-F6:FILE ACT 
  7 = ALT-F7:CODE             8 = ALT-F8:(no function) 
  9 = ALT-F9:LINE INS         0 = ALT-F10:LINE DEL 
  A = Tasks of function keys without ALT 
  F = Function keys F11 and F12 (Sucro keys) 
  K = Keys in general 

Information can also be obtained by AF1? AF2? etc.
