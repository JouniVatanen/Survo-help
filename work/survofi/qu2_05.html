TABD <table>,C,L
deletes classifier C.
In a frequency table the elements of the modified table are the
marginal sums of C-classification; In this case TABD means collapsing
over classifier C and the results is an (n-1)-dimensional table.
In a data table the C-classification will be changed to N-classifi-
cation where the C-class names are replaced by indices 1,2,...
The modified table is still n-dimensional and has the old structure
and contents; only the name C is replaced by N.
If TABD is applied to a data table which already includes an N-classi-
fication, the C-classification will be united to the N-classification
and now the dimension of the table decreases by 1 but the number of
elements remains the same.
