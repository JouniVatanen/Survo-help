Certain standard settings in the SURVO.APU file can be left out and
they are replaced by default settings as follows:

edisk (selects the default data path)
      Default is edisk=<Survo_path>\D\        (C:\E\D\, for example)
eout  (selects the ascii file for the results)
      Default is eout=<Survo_path>\TMP\RESULTS
qpath (selects the inquiry system)
      Default is qpath=<Survo_path>\Q\
tempdisk (selects the path for temporary files)
      Default is tempdisk=<Survo_path>\TMP\ .
