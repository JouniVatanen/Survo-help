Original PostScript does not have codes for accented characters (like
ä,ö) and other special text characters used in various languages, but it
has predetermined names for them (ä is adieresis, for example).
Each application system has to encode selected characters. The current
version of PS.DEV does so for all scandinavian characters (ä,ö,å,Ä,Ö,Å)
by giving them the same ASCII codes as used in the IBM PC.
The user can extend and alter the set of special characters by editing
PS.DEV. By listing this file, it is very easy to find the current setting
and alter the selection according to one's needs.
