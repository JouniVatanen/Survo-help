Exchange rates for currencies:
Example:
  100(USD:FIM)=431.8
In file <Survo>\U\SYS\MEASURES.EDT, each currency is represented by
two rates (Buy,Sell). The proper alternative is selected according
to the case. If only one value is given, it will be used as a rate
in both directions.
To update these rates, the easiest way is to use the UPDATE scheme
in the edit file <Survo>\U\SYS\CURRENCY.EDT .
