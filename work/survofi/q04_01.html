SAVE <edit file> 
saves the contents of the current edit field in file <edit file>.EDT
on the data disk. Also all display attributes of the text (like
underlining) are saved.
This is the normal procedure in saving edit fields.
Information about various edit file formats: See SAVEMM?
To load a field saved by SAVE use LOAD <edit file>.
To save an edit field in a standard text file, use COPY or SAVEP.
