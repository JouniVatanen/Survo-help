A control line (with a '-' in the control column) in the print list
or in an include file (driver) and having the form
  - codes <character_code_conversion_file> 
specifies the character conversion code for the PRINT operation.
See also CODES? 
If no - codes control line appear in the print list or in the include
files, no conversion takes place.
  P = More information on PRINT
