   - replace <string1> <string2> 
as a control line in the PRINT list replaces each occurrence of
<string1> in the text to be printed by <string2>. Only the latest
- replace line is valid.
This feature is useful in giving varying chapter names or numbers
on header lines typically in the form
   - replace ????? 3.1
