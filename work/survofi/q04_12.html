Help System (web edition)

The text in the edit field can be displayed in various colors by adding
shadow characters to selected characters on shadow lines.
A shadow line is created for the current edit line either by the PREFIX S
key or by the FORM key.
The FORM key gives the shadow values 0,1,2,3,4,5,6,7 in turn. The current
shadow value is displayed as the last character of the header line.
All characters typed will then have the selected shadow (color).
These shadow values usually have special tasks when the text is to be
printed (See PRINT? - effects of display attributes).
More alternatives can be created by using PREFIX S. This key reveals
the shadow line of the current edit line and the user can type and
edit characters of the shadow line. When PREFIX S is pressed again,
the shadow line disappears but its influence on the current line
will be in effect. Keysucro /S (activated by PREFIX M S) is
a handy tool for typing shadow characters.
The current number of shadow lines in use can be seen by activating /DIM .
A chart of all shadow colors is found by LOAD <Survo>\U\D\CRT

  S = SHADOW command (changing display attributes) 
  F = More information on the edit field
