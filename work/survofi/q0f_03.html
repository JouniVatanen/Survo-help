Help System (web edition)

COLOR ALL
displays all possible display modes (color combinations) in the Survo
main window. This is a help function for selecting colors for different
shadow values.

COLOR #16-31
(for example) displays the color combinations for values 16-31.
Values 0,1,2,...,255 are permitted.

  S = SHADOW command for changing shadow codes 
  C = More information on control operations
