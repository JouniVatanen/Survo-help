TEXTCOLS L1,L2,<W>,<L>,<C>,<text> 
transforms a text column on lines L1-L2 and having the width <W> 
to text blocks each of them having <C> columns and <L> lines.
Optional <text> is inserted as an extra line between the blocks.
Example on the next page:

TEXTCOLS CUR+1,CUR+16,5,4,3,/_New_page
1111
1111
1111
1111
2222 
2222
2222
2222
3333
3333

3333
4444

4444
This line and lines below it remain intact.
