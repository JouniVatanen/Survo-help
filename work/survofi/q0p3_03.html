The default setting (in PS.DEV) is a standard A4 sheet in upright
(portrait position). By [LANDSCAPE] the page will be rotated 90 degrees
to landscape orintation. Thereafter [PORTRAIT] will restore the original
portrait orientation. If [LANDSCAPE] is used for consecutive pages, it
is the user's responsibility to alter (diminish) the page length (set
by [page_length(x)] where x is given in points.)

In PRINT also control words [HOME(x,y)], [SCALING(kx,ky)], and
[ROTATION(angle)] can be used to achieve an arbitrary location, scaling,
and rotation of the output on the current page. Default settings are
[HOME(0,0)]     (physical left-bottom corner of the paper)
[SCALING(1,1)]
[ROTATION(0)]
