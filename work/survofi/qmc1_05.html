COMB S,L / S=SUBSETS,<n> 
lists all non-empty subgroups of <n> elements.
COMB S2,L / S2=SUBSETS,<n>,<m> 
lists subsets of <n> elements consisting of <m> elements.
Example:
COMB SUB3,CUR+1 / SUB3=SUBSETS,5,3
Subsets of size 3 of 5 elements: N[SUB3]=10
1 2 3
1 2 4
1 2 5
1 3 4
1 3 5
1 4 5
2 3 4
2 3 5
2 4 5
3 4 5

COMB S2,L / S2=SUBSETS,<n>,<m> ELEMENTS=<list_of_elements> 
lists all m-subsets of <n> elements given by ELEMENTS list with
possible repetitions.
Example:
COMB SUB5R,CUR+1 / SUB5R=SUBSETS,8,5 ELEMENTS=a,a,a,b,b,c,c,c
5-subsets of 8 elements (with repetitions): N[SUB5R]=9
a a a b b
a a a b c
a a a c c
a a b b c
a a b c c
a a c c c
a b b c c
a b c c c
b b c c c

    C = Other forms of COMB
