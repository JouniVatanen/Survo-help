Creating sound files from Survo data

There are two alternatives for making sounds based on Survo data.

1. Sounds from raw data:
Consecutive values of one (mono) or two (stereo) variables in a Survo
sound files can be saved as such in a WAV file by PLAY DATA command
and played back by PLAY SOUND commmand.

2. Mapping data values to given sounds:
Consecutive values of selected variables in Survo data files can be
replaced by sounds and "played" by
PLAY DATA <data> / VOICES=<list of given voices>.

Description of these alternatives not yet ready!
Examples (in Finnish) by
