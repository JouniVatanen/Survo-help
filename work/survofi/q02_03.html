Function keys:
+----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+
| F1 |  | F2 |  | F3 |  | F4 |  | F5 |  | F6 |  | F7 |  | F8 |  | F9 |  | F10|
|    |  |    |  |    |  |    |  |    |  |    |  |    |  |    |  |    |  |    |
+----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+  +----+
HELP    PREFIX  TOUCH   DISK    FORM    MERGE   REF     EXIT    INSERT  DELETE


  1 = F1:HELP(PREFIX)         2 = F2:PREFIX 
  3 = F3:TOUCH                4 = F4:DISK 
  5 = F5:FORM                 6 = F6:MERGE 
  7 = F7:REF                  8 = F8:EXIT 
  9 = F9:INSERT               0 = F10:DELETE 
  A = Tasks of function keys when ALT is pressed at the same time 
  F = Function keys F11 and F12 (Sucro keys) 
  K = Keys in general 

Information can also be obtained by F1? F2? etc.
