Display-off mode of Survo

In certain applications of Survo it is good to freeze the main window
and display only selected portions of the work on the screen. This
is appropriate especially in sucros made as teaching programs or
as presentations of graphical displays.
The key code PREFIX `L' `o' (in sucros {disp off}) disables the main window.
Text is written normally in the edit field but not shown in the window.
the main window is restored (and updated) by the keystrokes
PREFIX `L' `O' (in sucros {disp on}) or by the F8:EXIT key.

It is also possible to hide the main window by the WIN MIN command.
Then it is restored by WIN NORMAL.
Test this by continuous activation (F2 ESC) of the commands:
WIN MIN
WAIT 2
WIN NORMAL
