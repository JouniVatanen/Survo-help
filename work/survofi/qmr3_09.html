ROTATE <factor_matrix>,<number_of_factors>,L
with an optional ROTATION specification makes an orthogonal or oblique
factor rotation.
The following results are obtained as matrix files:
  AFACT.M Rotated factor matrix A
  TFACT.M Rotation matrix T
  RFACT.M Factor correlation matrix R

If RESULTS<=70, only A will be written to the edit field (from line L).
Otherwise, also T and R are printed to the edit field.

In case of an orthogonal rotation, A=F*T (F=<factor matrix>) and in
an oblique rotation A=F*INV(T').
Optionally, an initial rotation matrix can be given by T=<matrix_file>.
Thus results of automatic rotations can be used as starting points
for a more subjective graphical solution.
