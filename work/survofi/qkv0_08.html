/BARPLOT <data>                                          / K.Vehkalahti
plots a bar chart of <data> on the screen. In addition to the PLOT operation,
the colors of the bars can be selected individually (for special applications).
By default, a set of color intensities (CMYK) is given in a form of a data
table, which can be modified in various ways.

IND, CASES and SELECT specifications may be used for selecting observations.

Possible chart types are HBAR (default), VBAR, NHBAR and NVBAR, given by the
specification TYPE.

The sucro gives the user a ready made work scheme, where various parameters
can be modified, and the picture then easily re-produced. A PostScript
version of the picture can be created from that scheme by one sucro command
as well.
