Printer: See also PRINTER?
Parameter `printer' gives the printer port or name for the default printer.
Examples: printer=LPT1: printer="HP LaserJet 4M Plus"
    Parameter `print_dev' specifies the default device driver in the
PRINT operation. Example: print_dev=PS.DEV .
    Parameter `plot_mode' tells the default device in PLOT operations.
The DEVICE specification always dominates this setting. Example:
plot_mode=PostScript .
    Parameter `printdef' gives the maximum amount of bytes for
definitions of control words in the PRINT operation. Default is 60000.


Drivers for plotting:
The device drivers for PLOT operations are selected by parameters
ps_dev       for PostScript (ps_dev=PS.DEV)
crt_dev      for the screen (crt_dev=CRT16.DEV)
