CONVERT L1,L2,<conversion_file>,<column> 
converts all the characters on lines L1-L2 in the current edit field
according to the <column> (1,2,...) of <conversion_file>.
Default value for <column> is 1. Each column consists of 256 bytes.
A <conversion_file> can be defined by the user and saved by the
CODES SAVE operation (see CODES).
