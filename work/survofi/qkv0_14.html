/JITPLOT <data>,<xvar>,<yvar>                            / K.Vehkalahti
creates a scatter plot of two variables in the data set <data>, allowing
jittering of the values of either or both variables.

The jittering is based on the same principle as in the draftsman's display
(see DRAFTS?). In this sucro, the jittering may be set for both variables
independently by JITTER=kx,ky . For example, JITTER=0,ky leaves <xvar> 
intact, thus creating a 'dot plot'.

The true values are plotted as well, if a specification TRUE=1 is given.

The sucro gives the user a ready made work scheme, where various parameters
can be modified, and the picture then easily re-produced. A PostScript
version of the picture can be created from that scheme by one sucro command
as well.
