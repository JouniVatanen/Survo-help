TXT operations for text files

  1 = TXTCONV  for character and string conversions 
  2 = TXTRIM   for adjusting the line length of a text file 
  3 = TXTCOUNT for counting frequencies of various characters/line 
  4 = TXTEDTOUT for saving an edit field with shadows etc. in a text file 
  5 = TXTEDTIN  for loading an edit field saved by TXTEDTOUT 
  6 = TXTDEL   for deleting lines between two lines given in the edit field 
  7 = TXTSHORT for shortening long fields 

  T = More information on text editing
