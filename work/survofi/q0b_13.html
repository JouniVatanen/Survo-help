Size of the edit field:

The size of the edit field is controlled by parameters `ed1', `ed2' and
`ed3'. `ed1' is the field width (control character included), `ed2' is
the number of edit lines and `ed3' is the maximum number of shadow
lines. These values seldom have practical meaning since each edit field
loaded from disk resets their values (as the START field when Survo is
entered). The REDIM or INIT command is used to select new values for
`ed1', `ed2' and `ed3' for the current edit field.
Example: ed1=101 ed2=100 ed3=20 .
