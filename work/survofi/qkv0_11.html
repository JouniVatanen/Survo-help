/DATACOPY <original data> TO <target file>        / K.Vehkalahti
copies all the active fields in a Survo data set <original data> to a
new Survo data file, <target file>. The <original data> can be either
a data table in the edit field or a Survo data file.

The records to be copied may be limited by the IND, CASES and SELECT
specifications. The variables (fields) are pointed out either by MASK
or VARS specifications or using FILE ACTIVATE.

New space is allocated for the comment texts in <target file>.
Optionally more comments may be saved from the edit field, by giving
the corresponding lines by a specification LINES=L1,L2 .
The <target file> (a new data file) is always created. If it already
exists, overwriting is confirmed. To copy records to an existing data
file, use FILE COPY.

By combining the standard FILE operations of Survo, this sucro
provides an easy way for typical situations of data copying.
